Monster Hunter New World

Little Game on a Command Prompt

Project Description

This was a small project I made with 3 other persons during my first year at the IUT de Dijon. The goal was to make a little game on a command prompt with FreePascal. I made all the user interface part with its logic while my teammates made all the logic stuff and I love the final result a lot. Because we were on a command prompt, we only had characters to work with, so that's why we decided to use ASCII-art to illustrate the game ! Because I wanted it to look nice, I made a lot of them animated so it feels more alive. Also, menus scroll automatically to show all your items in your inventory. The ASCII-arts of the rooms and monsters were found on the internet, so thanks to the authors who made them ! ❤
