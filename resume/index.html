Python

Then, I discovered Python. I loved it because it's beginner friendly, while doing very complex stuff. It's pretty good at everything like making web backend (e.g. Flask), building desktop applications (e.g. PyQt, PySide), analysing data and making artificial intelligence (e.g. Panda, OpenCV), etc. Because it's an interpreted language, it's slower than compiled languages. However, it allows easy merge with other languages like C++ so it become very good for doing most of the complex work, while the really time consuming stuff is written in C++. It's a well balanced language in my opinion.
